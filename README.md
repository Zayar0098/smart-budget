# Smart Budget（スマート・バジェット）

Smart Budget は、**日本で生活する留学生やアルバイトをしている学生向け**に開発した、シンプルで使いやすい家計管理 Web アプリケーションです。収入・支出・貯金状況を月ごとに分かりやすく管理できることを目的としています。

---
## デモ・スクリーンショット
![Demo](appdemo.png)

## 機能・特徴
- **収入・支出の管理**：日々の収入や支出をカテゴリー別に記録
- **カテゴリー管理**：家賃・ガス代などの基本生活カテゴリーをあらかじめ用意し、追加も可能
- **自動合計計算**：月ごとの合計金額・残高を自動計算
- **数量指定入力**：カートのような感覚で金額を簡単に入力
- **月次確認**：当月の収支状況を一目で確認
- **ローカル保存**：`localStorage` を使用し、ログイン不要・オフライン対応
- **留学生向け設計**：日本語に不慣れなユーザーでも使いやすいUIを意識

## 使用技術

* **フレームワーク**：Next.js（App Router）
* **言語**：TypeScript
* **スタイリング**：CSS Modules
* **状態管理**：React Hooks
* **データ保存**：Browser `localStorage`

## ページ構成

* `/` ：ホーム（ダッシュボード）
* `/pincome` ：バイト収入管理
* `/month` ：月別収支確認
* `/saving-mode` ：予算・節約
* `/settings` ：設定画面

## インストール・セットアップ
### 前提条件
- Node.js 20.0以上
- npm 10.0以上

## インストール手順

### 1. リポジトリをクローン
```bash
git clone https://github.com/Zayar0098/smart-budget.git
cd smart-budget
```

### 2. 依存関係をインストール
```bash
npm install
```

### 3. 環境変数を設定
プロジェクトルートに `.env.local` ファイルを作成します：
**Windows の場合：**
```bash
type nul > .env.local
```

**Mac/Linux の場合：**
```bash
touch .env.local
```

`.env.local` ファイルを開いて、以下を追加：
```env
# Exchange Rate API のアクセスキー
NEXT_PUBLIC_EXCHANGE_RATE_API_KEY=あなたのAPIキー
```

 API キーの取得方法

1. [https://www.exchangerate-api.com/](https://www.exchangerate-api.com/) にアクセス
2. 「Get Free Key」ボタンをクリック
3. メールアドレスを登録
4. 受信したメールから API キーをコピー


### 4. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで以下の URL を開きます：
```
http://localhost:3000
```

### 5. 本番ビルド（任意）

本番環境用にビルドする場合：
```bash
npm run build
npm start
```
## データ管理について

* 金銭データはすべてブラウザ内に保存されます
* 月が変わると自動で初期化される仕組みを実装しています
* 外部サーバーへの送信は行いません
